# Demo Walkthrough

CodeShelf does not include a runtime demo mode. The app is intended to run against the developer's own GitHub account through a local `.env` token.

This page is a public placeholder for the future screenshot/GIF walkthrough.

## Walkthrough Outline

1. Start the local app with `npm run dev`.
2. Open `http://localhost:4200`.
3. Confirm the setup/status screen shows GitHub and AI provider state.
4. Load repositories from the local backend.
5. Review dashboard stats and public/private repo sections.
6. Use search, sorting, and suggestion filters.
7. Review local portfolio and cleanup scores.
8. Run AI analysis if configured. Only public repos are sent to the backend AI route.
9. Select one or more repos for a visibility change.
10. Review the warning screen before confirming.
11. For deletion, enable the deletion safety toggle, manually select repos, confirm, and review the result report.
